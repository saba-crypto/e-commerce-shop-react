# OS-game-store-react
## this repository showcases more advanced version of OS-game-store project built using React and Express, I built simple backend with simple API endpoints. (backend works without database, saving any backend code will re-start the server)
### Note: to start both frontend and backend servers, type "npm start" when you are at "ecommerce-backend" folder and "npm run dev" for "ecommerce-frontend" folder as well.

### technologies that I use for this project:
* HTML
* CSS(TailwindCSS)
* TypeScript
* React-Vite
* Node.js
* Express
* Git(GitHub)
